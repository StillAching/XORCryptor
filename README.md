# XORCryptor
This is a basic implementation meant for educational purposes, showcasing how XOR encryption works.

# Usage:
Example:

XOR Encryption:

Input: HelloWorld

Key: key123

Output: A hex-encoded string (e.g., 3b16104a1c1a4a061414)

---------------------
XOR Decryption:

Input: 3b16104a1c1a4a061414

Key: key123

Output: HelloWorld
